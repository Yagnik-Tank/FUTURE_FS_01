
Personal Portfolio Website

A modern and responsive Personal Portfolio Website built using React + Vite. This project showcases
personal information, skills, projects, and contact details in a professional and attractive layout.

🚀Features
Responsive modern UI design
Smooth navigation and sections
Hero section with introduction
About section
Skills section
Projects showcase
Contact section
Fast performance using Vite
Clean and reusable React components

🛠️Tech Stack
React.js
Vite
JavaScript (ES6+)
CSS3
HTML5

⚙️Installation & Setup
1. Clone the repository:
git clone https://github.com/Yagnik-Tank/FUTURE_FS_01.git

2. Navigate to the project folder:
cd personal-portfolio

3. Install dependencies:
npm install

4. Run the development server:
npm run dev

5. Open in browser:
http://localhost:5173
